# HƯỚNG DẪN CÀI ĐẶT

Để cài đặt dự án Nha Nước Remote, hãy làm theo các bước sau:

1. Clone repository:
   ```bash
   git clone https://github.com/your_username/nha-nuoc-remote.git
   cd nha-nuoc-remote
   ```
2. Cài đặt các phụ thuộc:
   ```bash
   npm install
   ```
3. Chạy ứng dụng:
   ```bash
   npm start
   ```

Chúc bạn thành công!
